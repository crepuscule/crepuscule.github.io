工作区和暂存区

本文中，工作区即GitLearn.版本库即为.git，里面有stage(index)暂存区，和分支链表。git再初始化时会自动创建master分支，指向当前版本的链表指针即为HEAD。
git add 将文件添加进入stage，然后git commit将stage中的更改一次性地添加到分支链表。

当每次修改文件完成时，均需要git add才能commit。
