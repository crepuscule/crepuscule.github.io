for

Python 的 for 语句依据任意序列（链表或字符串）中的子项，按它们在序列中的顺序来进行迭代

需要注意的是，在迭代过程中修改迭代序列不安全，可以使用其副本进行迭代

range()

生成一个等差的链表序列，如：
for i in range(5): print(i)
结果输出 0~4 5个数字
这样我们可以用range()实现从自定义的数值
