开启/关闭SELINUX的方法

  1. 临时开启/关闭

在启动之前可以修改Linux核心参数(Kernel Parameter) 或者可以在核心参数后加上： selinux=0 （停止） 或 selinux=1 (开启）参数
/boot/grub/menu.lst

在机器运行时可以直接用命令setenforce修改

  1. 永久关闭
    root权限修改/etc/selinux/config文件中 SELINUX=disabled，千万不要修改下面的 SELINUXTYPE=targeted
