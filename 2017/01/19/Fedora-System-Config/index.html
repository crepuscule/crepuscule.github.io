LVM

LVM即Logical Volume Manager（逻辑卷管理），是linux下对磁盘管理的一种机制。与普通磁盘分区管理机制不同的是，它可以在零停机前提下实现对分区大小的调整，其分区不需要必须连续的空间，所以可以跨越不同的磁盘和分区。

首先需要了解一些LVM术语：

  • 物理卷 PV （Physical Volume）
    指磁盘分区或从逻辑上与磁盘分区具有同样功能的设备（如RAID），是LVM的基本存储逻辑块，但和基本的物理存储介质（如分区、磁盘等）比较，却包含有与LVM相关的管理参数
  • 物理块 PE （Physical Extent）
    每一个物理卷PV被划分为称为PE（Physical Extents）的基本单元，具有唯一编号的PE是可以被LVM寻址的最小单元。PE的大小是可配置的，默认为4MB。所以物理卷（PV）由大小等同的基本单元PE组成
  • 逻辑块 LE （Logical Extent）
    逻辑卷LV也被划分为可被寻址的基本单位，称为LE。在同一个卷组中，LE的大小和PE是相同的，并且一一对应
  • 卷组 VG （Volume Group）
    类似于非LVM系统中的物理磁盘，其由一个或多个物理卷PV组成。可以在卷组上创建一个或多个LV（逻辑卷）
  • 逻辑卷 LV （Logical Volume）
    类似于非LVM系统中的磁盘分区，逻辑卷建立在卷组VG之上。在逻辑卷LV之上可以建立文件系统（比如/home或者/usr等）
    LVM就是允许不连续的PV组成一个逻辑上的磁盘（即VG），然后在VG上实现类似非LVM的磁盘分区，显然这样在逻辑上划分分区就更加灵活。

常用命令:

  • 物理卷 pv

  • 卷组 VG

  • 逻辑卷 LV

RPM Fusion库

由于版权问题，许多软件在Fedora中是没有的，如果想安装，需要添加RPM Fusion库,有两个RPM Fusion提供：免费和非免费。

Fedy Tool

Fedy Tool可以让你安装多媒体和附加软件，如MP3、Adobe Flash、java等。

安装成功完成，启动Fedy Tool。在应用程序选项列表中点击需要安装的软件。我们也可以安装常用的软件包像adobe flash player, Drop Box, Google Chrome, Multimedia codecs, skype, steam, oracle java, telegram, Wps office等。
