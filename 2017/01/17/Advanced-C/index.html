C中define的用法

  1. 简单用法
    #define MAX 5+5
    在预编译阶段在程序中发现MAX即将其替换为5+5, 不会对宏体做任何运算。
    如果有语句int a=NN;a的值应为5+55+5=35
  2. 带参数的宏定义
    #define add(x,y) x+y
    如果要用define实现一个加法函数，上述做法是错误的，正确做法是：#define add(x,y) ((x)+(y)) 如果用宏实现类似函数的功能，必须将每一个参数和整个宏体都加上括号。
  3. 多行宏定义
    如同多数命令行一样，在行末尾加一个反斜杠”\”，即可在下一行继续书写宏定义。
  4. 宏中三个符号
  • x##y 表示连接 #define con(x,y) x##y int n=con(4,5); n的值为45
  • #@x 表示给x加上’’,使其变为一个const char
  • #x 表示给x加上””,使其变为一个字符串

注意，#@是微软的编译器专属，对于gcc没有这个用法
