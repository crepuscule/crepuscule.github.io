fedora常用软件的安装

Input Methods

Rime(中州韵)

  • ibus-rime
    fedora自带的输入法管理器是ibus，个人认为gnome桌面下还是老老实实用ibus比较好，在gonme桌面下无论安装还是配置均是ibus方便，而且安装fcitx后会导致输入法切换很头疼。

  • fcitx-rime
    因为有了fedora中文社区软件源安装fcitx-rime也很方便，添加了fedora中文社区软件源后直接终端输入

搜狗输入法

fedora下安装搜狗输入法只能使用fcitx管理输入法
同样，在fedora中文社区软件源也有它的rpm包

Programming

atom

  • 使用 Fedy 安装
    Fedy Tool是一个可以让你安装多媒体和附加软件的工具，可以轻松地安装MP3插件、Adobe Flash、java等在fedora较为难装的软件。

    安装完成之后，直接在图形界面找到atom安装即可。

  • rpm包本地安装
    在github上下载atom.x86_64.rpm包，然后运行

网易云音乐

  • 网易云音乐GUI版
    网易云音乐官网是没有rpm包的，但是有很多fedoraer将deb包转换成了rpm包以供fedora系统使用。安装前需要已经添加rpm fusion源。

    使用此链接或者在此链接中选择，下载网易云的rpm包。
    然后本地安装该rpm包

    值得一提的是，如果你的fedora系统没有安装MP3插件，网易云也是无法播放音乐的

  • 网易云音乐命令行版
    如果你嫌安装图形版的网易云繁琐，可以试试它的命令行版本。不仅安装方便，而且使用简洁快速，逼格甚高。该版本在fedora中文社区软件源中就有，你可以直接下载rpm包然后本地安装，也可以添加fedora中文社区的源，使用dnf命令安装。

  • 尝试深度音乐
    据说深度音乐和网易云音乐差不多，如果不喜欢命令行版本的也可以尝试深度音乐，fedora中文社区软件源有提供。

NetWork

ss-qt5

关于ss-qt5在github ss-qt5 wiki上有详细说明。

  • Fedora下安装
    目前支持RHEL 7, Fedora

  • Ubuntu下安装
    通过PPA源安装

  • Debian下安装

    • methed1 尝试安装Ubuntu PPA源的deb包，
    • methed2 自行编译（dpkg-buildpackage -uc -us -b），在上级目录中将会生成shadowsocks-qt5的deb包，通过sudo dpkg -i来安装。

      注意：你可能需要安装好的依赖关系：sudo apt-get install qt5-qmake qtbase5-dev libqrencode-dev libqtshadowsocks-dev libappindicator-dev libzbar-dev libbotan1.10-dev
